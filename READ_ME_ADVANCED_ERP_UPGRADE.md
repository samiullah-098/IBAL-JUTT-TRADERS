# Advanced ERP Upgrade by ChatGPT

This package adds:

1. Easy Urdu guide on every page.
2. Dashboard date range filters with daily/weekly/monthly/custom summaries.
3. Global topbar search removed.
4. Brand name locked as IQBAL JUTT TRADER in Settings.
5. Expenses module with add/edit/delete/detail.
6. Supplier Quotation Generator with inventory item selection and WhatsApp/share-ready text.
7. General Ledger screen covering sales, purchases, payments, expenses and staff salary payments.
8. Security login screen with admin username/password/PIN settings.
9. Staff module with profile details, salary payments, login credentials and section permissions.
10. Mobile-friendly layout improvements and improved receipt print design.

Default first login:
Username: admin
Password: admin
PIN: optional or 1234

Run:
backend: npm install && npx prisma generate && npm run dev
frontend: npm install && npm run dev

Important:
After logging in, open Settings and change admin username/password/PIN.
